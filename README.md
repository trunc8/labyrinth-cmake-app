# labyrinth-cmake-app
*Developed as part of JdeRobot's [GSoC C++ Challenge](gsoc2021-c++-test.pdf)*

### Usage

#### Modifying labyrinth

Please edit the file `labyrinth.txt`

#### Running the solution
```
mkdir build && cd build
cmake .. && cmake --build . && ./app
```
